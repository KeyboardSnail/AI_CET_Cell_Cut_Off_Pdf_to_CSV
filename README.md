## AI_CET_Cell_Cut_Off_Pdf_to_CSV

This Python script converts cutoff PDF files from the CET Cell website into CSV format, enabling seamless data analysis.
A command-line tool built with click for CLI interface, pathlib and os for robust file handling, and pdfplumber alongside PyPDF2 for accurate text extraction from tabular PDFs. It employs re (regular expressions) to parse structured cutoff data—such as ranks, scores, categories, and rounds—into clean DataFrames, then exports to CSV for easy import into pandas, Excel, or SQL tools.
